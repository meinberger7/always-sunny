# AGENTS.md

Project instructions for Codex contributors:

- Start each task from the latest `main` branch before making changes.
- Create a **new branch** and a **new PR** for every task.
- Keep each PR small, focused, and single-purpose.
- Do not change app behavior unless the task explicitly asks for it.
- Do not change **Harvestable mode** unless explicitly requested.
- Do not change deployment, CI, workflows, or GitHub Pages setup unless explicitly requested.
- Preserve the frontend data contract unless explicitly asked to change it.
- Explain PR changes in plain English (what changed, why, and impact).
- Prefer simple, readable code over clever or complex solutions.
- This repo uses **GitHub Pages from `main`** as the live test environment.
- Assume the repo owner is a novice GitHub user; keep instructions low-jargon and practical.

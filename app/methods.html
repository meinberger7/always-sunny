Methods & Caveats

This dashboard highlights geographic diversity of wind and solar output. It does not model all reliability constraints.

Harvested mode

Uses balancing-authority-visible hourly values (utility-scale-visible perspective). Values in this MVP are sample data scaffolded to match intended EIA-930-style schema.

Harvestable mode

Shows scenario-based modeled estimates from precomputed demo data. Transmission/storage assumptions influence how diversity can be used:

  • Local: buildout constrained inside each BA.
  • Regional: buildout optimized within each interconnection.
  • National: buildout optimized nationwide under macrogrid assumption.

Future versions will replace demo scenarios with weather-driven precomputed outputs linked to supply-curve economics.
